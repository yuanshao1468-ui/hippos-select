# Shopee 導購商城專案 (Shopee Store)

這是一個由 Jim 老闆主導、小蝦全自動維護的 **「質感極簡風」蝦皮導購網站**。

## 專案目標
- 透過「走量模式」引擎自動抓取蝦皮高銷量、大眾款商品。
- 每天凌晨自動更新 JSON 資料庫。
- 採用 Vercel 部署，提供極致的手機端購物體驗。
- 將公域流量（LINE/FB/短影音留言區）洗入私域網站。

## 技術架構 (規劃中)
- 前端框架: React 或 Vanilla JS + Tailwind CSS (輕量、載入快)
- 資料來源: 每日產出的 `store_data.json`
- 部署平台: Vercel 或 GitHub Pages
